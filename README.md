# Simula Urna


Install Node
```bash
$ sudo apt install nodejs
$ nodejs -v
```
Install NPM
```bash
$ sudo apt install npm
$  npm -v
```

Install Ionic

```bash
$ npm install -g @ionic/cli
```


To deploy this project run

```bash
# Clone the repo
$ git clone https://github.com/mirandastephane/simula-urna.git

# change directory to our repo
$ cd simula-urna

# start
ionic serve
```
